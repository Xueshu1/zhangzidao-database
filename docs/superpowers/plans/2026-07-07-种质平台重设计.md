# 种质平台重设计 实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把 `index.html` 从"电影感滴动叙事页"重构为"专业种质资源数据平台"——固定顶栏 + hash 路由多视图 + 每页科技背景 + 升级排版,A 严肃骨架 + B 炫技视觉。

**Architecture:** 单文件 `index.html`,原生 JS(非框架)。用 hash 路由在同一外壳内切换 7 个视图(#home/#resources/#databases/#analysis/#search/#standards/#about)。全站固定分层背景栈(粒子场 + CSS mesh + 蓝图网格 + 噪点)置于 z-index:0,内容坐半透明毛玻璃面板之上,每视图共享同一背景。复用现有 SPECIES/RESERVE 数据常量与 barChart/heatmap 图表助手。

**Tech Stack:** HTML + 原生 JS(ES5 风格,与现有代码一致)+ CSS 变量 + Three.js(粒子场,CDN)+ GSAP/ScrollTrigger(入场淡入,CDN)。无构建工具、无 npm、无测试框架。

## Global Constraints

- **单文件**:所有代码在 `index.html` 内;仅图片在 `assets/species/*.jpg`。
- **离线可用**:双击即用,无需服务器。CDN 脚本失败时核心内容仍可见(渐进增强)。
- **DEMO 数据**:所有样本数/基因组指标/位点为演示示例数据,保留免责声明。
- **中文优先**:保留地理专名(如"北黄海冷水团")。
- **配色变量**:沿用 `:root` 里 CSS 变量与 `ACC{}` 常量(cyan/red/purple/teal/steel/amber/rose),深海黑蓝基色 `#070b14` 系,不用纯黑。
- **可读性**:正文 ≥16px/400,标题 600-700,统计用等宽数字(font-variant-numeric:tabular-nums),对比度 ≥4.5:1。
- **动效纪律**:只动 transform/opacity/filter;`prefers-reduced-motion` 下全关。
- **无"差异性"残留**:导航/命令/JS 映射里"差异性"一律删除或升级为"遗传分析"。
