# 黄渤海海珍品 · 种质资源数字平台

> 獐子岛集团 × 辽宁省海洋动物免疫学与疫病防控重点实验室

覆盖 **虾夷扇贝 · 仿刺参 · 香螺 · 海胆 · 皱纹盘鲍 · 长牡蛎** 六大主营品种，兼顾战略储备品种，整合**活体库、基因库重测序、SNP 库与群体分化分析**四层数据资产的黄渤海海珍品种质资源数字平台。

## 在线预览

启用 GitHub Pages 后，可直接访问：

**https://xueshu1.github.io/zhangzidao-database/**

## 本地打开

本站为**单文件成品**，无需服务器、无需构建，双击即用：

```
直接用浏览器打开 index.html
```

建议使用 Chrome / Edge 等现代浏览器以获得最佳视觉效果（液态玻璃、SVG 图表、动效）。

## 功能一览

- **物种矩阵** — 六大主营品种卡片，点击查看完整种质基因组指标（照片 + 按染色体 SNP 分布图 + MAF 分布图）。
- **战略储备品种** — 面向未来良种选育的储备品种大图卡片。
- **四大数据库** — 活体库 / 基因库·重测序 / SNP 库 / 群体分化分析，各带内联 SVG 图表。
- **群体分化单元** — 四海区 Fst 热力图与溯源分析（并入仿刺参弹窗）。
- 纯原生 JS、零依赖、可离线运行。

## 文件结构

```

├─ platform (1).html   ← 备份：原平台基座

```

## 说明
