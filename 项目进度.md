# 獐子岛种质库网站 · 项目进度

> 最后更新：2026-07-08（多样化图表系统 + 丰富数据维度）

## 一句话
把一个复刻的电影感视频落地页，本地化并合并成**黄渤海海珍品·种质资源数字平台**的完整单文件成品站，每个按钮可点、每张卡有真实照片、每个数据库有图表。

## 🎨 最新更新（2026-07-08）
### ✅ 多样化图表系统与科学配色
- **新增图表类型**：
  - 棒棒糖图（Lollipop Chart）：用于展示群体遗传学指标（π、LD、Ne、Fis）
  - 圆环图（Donut Chart）：用于展示染色体SNP分布
  - 多色渐变柱状图：增强视觉美观性
  - 折线图、雷达图（已有，保留）
  
- **科学配色方案**：
  - 引入 Nature/Science 风格配色（ocean/teal/green/orange/purple等16色）
  - 自动多色组合：每个数据点使用不同颜色，避免单调
  - 渐变效果：支持线性渐变，增强立体感
  
### ✅ 补充数据维度
- **群体遗传学指标**（已添加到每个品种）：
  - 核苷酸多样性 π（2.56×10⁻³ ~ 4.82×10⁻³）
  - 连锁不平衡衰减距离 LD（8.6 ~ 24.8 kb）
  - 有效群体大小 Ne（18,400 ~ 56,400）
  - 近交系数 Fis（-0.018 ~ 0.028）
  
- **生长曲线数据**（月龄 vs 体重/壳长）：
  - 虾夷扇贝：3-24月，2.8g→128.6g
  - 仿刺参：4-30月，8g→240g
  - 皱纹盘鲍：6-42月，1.8g→145.2g + 壳长15mm→128mm
  - 长牡蛎：3-18月，4.2g→138.2g + 壳长24mm→108mm
  - 香螺、海胆（马粪/紫海胆）
  
- **环境适应性数据**（雷达图展示）：
  - 温度耐受范围（0~32℃）
  - 盐度适应范围（18~38‰）
  - 溶氧需求范围（4~16 mg/L）
  - 抗病指数（62~82分）

## 关键技术备忘
- 全站单文件 `index.html`，原生 JS（非 React）；顶部保留原平台 loader/粒子/顶栏/命令栏。
- 配色走 CSS 变量与 `ACC{}` 常量（cyan/red/purple/teal/steel/amber/rose）。
- 照片路径相对 `assets/species/<key>.jpg`，`key` 见 `SPECIES[]`（abalone/cucumber/oyster/scallop/urchin_h/urchin_p/whelk）。
- 所有数据均为 **DEMO 示例数据**，页面已标注免责声明。
- 验证技巧：无头 Edge 截图会卡在视频 preload 的 loader，用 `--dump-dom` 检查渲染更可靠。
